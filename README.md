# US-CN Legal Term Mapping

> 美国法 ↔ 中国法 · 法律术语映射表  
> 版本：2026.05.24 · 覆盖 21 个法律领域 · 890+ 条款引用

---

## 概述

本项目提供一份**系统性、可机器执行的**中美法律术语映射表，按中国部门法体系编排。核心用途：

- **术语替换** — 美国法术语 → 中国法标准术语（含【删除】标记项）
- **法条映射** — 美国法规 → 对应中国法条（均使用正式全称）
- **引注格式转换** — 美国引用格式 → 中国引用格式
- **审查点注入** — 中国法特有审查要点

## 法条引用规范

所有中国法引用使用正式全称，例如：

- 《中华人民共和国民法典》而非《民法典》
- 《中华人民共和国刑事诉讼法》而非《刑诉法》
- 《中华人民共和国个人信息保护法》而非《个保法》
